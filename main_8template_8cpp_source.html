12 /* This is a scaffold for a stand-alone AMICI simulation executable
13  * demonstrating the use of the AMICI C++ API.
14  *
15  * This program reads AMICI options from an HDF5 file, prints some results
16  * and writes additional results to an HDF5 file. The name of the HDF5 file
17  * is expected as single command line argument.
18  *
19  * An initial HDF5 file with the required fields can be generated using MATLAB
20  * by adding the following lines
21  * at the end of simulate_${MODEL_NAME}.m file just before the final "end":
22  *
23  * %% Write data that is passed to AMICI to HDF5
24  * hdffile = fullfile(pwd, 'mydata.h5');
25  * structToHDF5Attribute(hdffile, '/options', options_ami);
26  * h5writeatt(hdffile, '/options', 'ts', tout);
27  * h5writeatt(hdffile, '/options', 'nt', numel(tout));
28  * h5writeatt(hdffile, '/options', 'theta', theta);
29  * h5writeatt(hdffile, '/options', 'kappa', kappa);
30  * if(~isempty(data))
31  * structToHDF5Attribute(hdffile, '/data', data);
32  * end
33  *
34  * ... and then running a simulation from MATLAB as usual.
35  *
36  * Default UserData settings can be written to an HDF5 file with:
37  * structToHDF5Attribute('test.h5', '/options', amioption())
38  */
