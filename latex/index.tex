\hypertarget{index_autotoc_md1}{}\doxysubsubsection{About}\label{index_autotoc_md1}
AMICI provides a multi-\/language (Python, C++, Matlab) interface for the \href{https://computing.llnl.gov/projects/sundials/}{\texttt{ SUNDIALS}} solvers \href{https://computing.llnl.gov/projects/sundials/cvodes}{\texttt{ CVODES}} (for ordinary differential equations) and \href{https://computing.llnl.gov/projects/sundials/idas}{\texttt{ IDAS}} (for algebraic differential equations). AMICI allows the user to read differential equation models specified as \href{http://sbml.org/}{\texttt{ SBML}} or \href{http://pysb.org/}{\texttt{ Py\+SB}} and automatically compiles such models into Python modules, C++ libraries or Matlab {\ttfamily .mex} simulation files.

In contrast to the (no longer maintained) \href{https://computing.llnl.gov/projects/sundials/sundials-software}{\texttt{ sundials\+TB}} Matlab interface, all necessary functions are transformed into native C++ code, which allows for a significantly faster simulation.

Beyond forward integration, the compiled simulation file also allows for forward sensitivity analysis, steady state sensitivity analysis and adjoint sensitivity analysis for likelihood-\/based output functions.

The interface was designed to provide routines for efficient gradient computation in parameter estimation of biochemical reaction models but it is also applicable to a wider range of differential equation constrained optimization problems.\hypertarget{index_autotoc_md2}{}\doxysubsubsection{Current build status}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item SBML import (see details below)
\item Py\+SB import
\item Generation of C++ code for model simulation and sensitivity computation
\item Access to and high customizability of CVODES and IDAS solver
\item Python, C++, Matlab interface
\item Sensitivity analysis
\begin{DoxyItemize}
\item forward
\item steady state
\item adjoint
\item first-\/ and second-\/order
\end{DoxyItemize}
\item Pre-\/equilibration and pre-\/simulation conditions
\item Support for \href{https://academic.oup.com/bioinformatics/article/33/7/1049/2769435}{\texttt{ discrete events and logical operations}} (Matlab-\/only)
\end{DoxyItemize}\hypertarget{index_autotoc_md4}{}\doxysubsubsection{Interfaces \& workflow}\label{index_autotoc_md4}
The AMICI workflow starts with importing a model from either \href{http://sbml.org/}{\texttt{ SBML}} (Matlab, Python), \href{http://pysb.org/}{\texttt{ Py\+SB}} (Python), or a Matlab definition of the model (Matlab-\/only). From this input, all equations for model simulation are derived symbolically and C++ code is generated. This code is then compiled into a C++ library, a Python module, or a Matlab {\ttfamily .mex} file and is then used for model simulation.

\hypertarget{index_autotoc_md5}{}\doxysubsubsection{Getting started}\label{index_autotoc_md5}
The AMICI source code is available at \href{https://github.com/AMICI-dev/AMICI/}{\texttt{ https\+://github.\+com/\+AMICI-\/dev/\+AMICI/}}. To install AMICI, first read the \href{http://amici-dev.github.io/AMICI/md__i_n_s_t_a_l_l.html}{\texttt{ installation instructions}}.

To get you started with Python-\/\+AMICI, the best way might be checking out this \href{https://github.com/AMICI-dev/AMICI/blob/master/python/examples/example_steadystate/ExampleSteadystate.ipynb}{\texttt{ Jupyter notebook}}.

To get started with Matlab-\/\+AMICI, various examples are available in \href{https://github.com/AMICI-dev/AMICI/tree/master/matlab/examples}{\texttt{ matlab/examples/}}.

Comprehensive documentation on installation and usage of AMICI is available online for the \href{https://amici.readthedocs.io/en/latest/}{\texttt{ python}} and \href{http://amici-dev.github.io/AMICI/}{\texttt{ MATLAB/\+C++}} interfaces.

Any \href{http://amici-dev.github.io/AMICI/md__c_o_n_t_r_i_b_u_t_i_n_g.html}{\texttt{ contributions}} to AMICI are welcome (code, bug reports, suggestions for improvements, ...).\hypertarget{index_autotoc_md6}{}\doxysubsubsection{Getting help}\label{index_autotoc_md6}
