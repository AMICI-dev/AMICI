\doxysubsubsection{Detailed Description}
This class is used to provide separate AMICI contexts, for example, for use in multi-\/threaded applications where different threads want to use AMICI with different settings, such custom logging functions.

NOTE\+: For this moment, the context object needs to be set manually to any \mbox{\hyperlink{classamici_1_1_model}{Model}} and \mbox{\hyperlink{classamici_1_1_solver}{Solver}} object. If not set, they will use the default output channel.
