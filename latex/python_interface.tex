\hypertarget{python_interface_autotoc_md39}{}\doxysubsubsection{Constants}\label{python_interface_autotoc_md39}

\end{DoxyVerb}
\hypertarget{python_interface_autotoc_md40}{}\doxysubsubsection{Observables}\label{python_interface_autotoc_md40}

\hypertarget{python_interface_autotoc_md42}{}\doxysubsection{Model Compilation}\label{python_interface_autotoc_md42}

Note\+: To build A\+M\+I\+CI with Open\+MP support, which allows to parallelize model simulations of multiple experimental conditions, set the environment variables {\ttfamily A\+M\+I\+C\+I\+\_\+\+C\+X\+X\+F\+L\+A\+GS} and {\ttfamily A\+M\+I\+C\+I\+\_\+\+L\+D\+F\+L\+A\+GS} to the correct Open\+MP flags of your compiler and linker, respectively.
