AMICI

A Matlab Interface to CVODES and IDAS

View on GitHub Download .zip Download .tar.gz

AMICI is a MATLAB interface for the SUNDIALS solvers CVODES (for ordinary differential equations) and IDAS (for algebraic differential equations). AMICI allows the user to specify differential equation models in terms of symbolic variables in MATLAB and automatically compiles such models as .mex simulation files. In contrast to the SUNDIALSTB interface, all necessary functions are transformed into native C code, which allows for a significantly faster compilation.

The interface was designed to provide routines for efficient gradient computation in parameter estimation of biochemical reaction models but is also applicable to a wider range of differential equation constrained optimization problems.

Features

  • generation of model specific native C code
  • first and second order forward sensitivity analysis
  • adjoint sensitivity analysis
  • steady state sensitivity analysis
  • time-to-event sensitivities
  • parameter dependent discontinuities

Requirements

AMICI is a software package for MATLAB. It requires the Symbolic Math Toolbox and a correctly configured C Compiler.
